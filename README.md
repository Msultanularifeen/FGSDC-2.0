# FG Science Degree College (Redesigned)

A modern, high-performance web platform for FG Science Degree College, designed to bridge the gap between traditional education and the digital future. This project features a cutting-edge UI, integrated AI student assistance, and a comprehensive administrative portal.

- **Modern Minimalist Aesthetic**: Inspired by industry leaders (Vercel, Stripe), utilizing a deep "Zinc" palette with sophisticated glassmorphism effects.
- **AI-Powered Assistant**: Integrated Gemini 2.5 Flash chatbot to answer student queries regarding admissions, syllabus, and campus life instantly.

## üì¶ Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/fgsdc-redesign.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Set up environment variables:
    Create a `.env` file and add your Google Gemini API Key:
    ```
    API_KEY=your_api_key_here
    ```
4.  Run the development server:
    ```bash
    npm start
    ```
